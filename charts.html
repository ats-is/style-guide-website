  Charts

Display data in responsive charts using the Flot Charts library.

Note:   you must use the ats-full.min.js file to use charts. The charting libraries are not provided in the main javascript file due to flot's file size.   You must also include the ie.min.js script for IE8 support.

For additional help creating Flot Charts view the tutorials on This Website.

Go to Flot Charts

Dounut Charts

In addition to flot charts, the style guide also includes Easy Pie Charts for displaying stylish dounut charts. It's mostly useful for displaying percentages.

Demo

86

Html

<span class="chart" data-percent="86">
    <span class="percent">86</span>
</span>

Javascript

$(function() {
    $('.chart').easyPieChart({
        //other options here
        animate: 2000
    });
});
