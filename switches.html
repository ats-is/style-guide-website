  Toggle Switches

The style guide uses the Bootstrap Switch plugin to turn form inputs into toggle switches.

There are many more options available for these switches that are documented on the Bootstrap Switch website.


Toggle Switches 
<input type="checkbox" checked class="switch" data-on="success" data-switch="switch">
<input type="checkbox" checked data-on="success" data-off="info" data-switch="switch">
<input type="checkbox" checked data-on="success" data-off="warning" data-switch="switch">
<input type="checkbox" checked data-on="success" data-off="danger" data-switch="switch">
<input type="checkbox" checked data-on="success" data-off="default" data-switch="switch">
<input type="checkbox" checked data-on="success" data-off="info" data-switch="switch">
